数据库

层次模型：

网状模型：
关系型数据库： 能找到关系

要用管理员身份打开终端

字符编码：在Windows中给gbk
在linux中给UTF-8

仓库-》数据库
表：仓库下面，其实，有其他东西，归类就叫表。

id name age 字段；
auto_increment 自动增长；
primary key 主键 最主要的 就靠它来区分学生这张表；这个字段是唯一的，关系型数据库中的表与表之间的联系
comment 注释

DDL data definition language 数据库定义语言 create alter drop
show //操纵库和表用的
DML data manipulation language 数据库操纵语言 insert update delete select //数据操作用的
DCL //数据库管理

中文显示不正常时

存钱用decimal ,定点数，每几个数用几个字节，一边存小数，一边存整数。

varchar 变长字符串，会自动回收，节省时间
char 定长字符串，效率更高
text 长文本数据

auto_increment一定是主键，主键不一定是自增长；
好处：保证数据的完整性，加快查询某种数据的速度。
主键不能为NULL，有auto时可以，会自动填。

对于自增长的主键，与上面主键外键都不兼容。

置空：留给外界删除数据，删除主表id，在关联的表上id设为NULL
级联：留给外界更新数据，删除主表数据，在关联表上也删。更新数据的时候，把关联表上的数据也进行更新。
