## 泛型是什么
泛型实现了类型参数化，达到代码重用的目的
泛型相当于类型占位符
定义类或方法时使用替代符代表变量类型
当真正使用类或方法时再具体指定类型


## 泛型分类
泛型类和泛型接口：基本语法
```
class 类名<泛型占位字母>
interface 接口名<泛型占位字母>
```
泛型函数：基本语法
```
函数名<泛型占位字母>（参数列表）
```
泛型占位字母可以有多个，用逗号隔开

## 泛型类和接口
被申明和继承的时候<>里的内容都要被填写出来。
```C#
class TestClass<T>{
	public T value;
}

class Program 
{ 
	static void Main(string[] args)
	{ 
		Console.WriteLine("泛型");
		// 这里的T决定value的数据类型。
		TestClass<int> t = new TestClass<int>();
		t.value = 10;
		Console.WriteLine(t.value);
	}
}

interface TestInterface<T>
{ 
	// 这里的get和set方法是接口定义的属性。接口中的属性声明只包含访问器（`get`和`set`），而没有实现。实现类必须同时实现这两个访问器。
	T value
	{ 
		get;
		set;
	}
}

class MyClass : TestInterface<double>
{
    private double _value;
    public double value
    {
        get { return _value; }
        set { _value = value; }
    }
}
```

## 泛型方法
- [ ] 普通类中的泛型方法
```C#
class Test2{
	// 做参数的
	public void TestFun<T>(T value){ 
		Console.WriteLine(value);
	}

	// 不做参数的
	public void TestFun<T>(){ 
		// 用泛型类型做一些逻辑处理
		T t = deFault(T);
	}

	// 作为返回值
	public T TestFun<T>(string v){ 
		return default(T);
	}
}

class Program 
{ 
	static void Main(string[] args)
	{ 
		Console.WriteLine("泛型");
		Test2 tt = new Test2();
		// 这里的<string>可以省略，推荐加上
		tt.TestFun<string>("123456")
	}
}
```
- [ ] 泛型类中的泛型方法
```C#
class Test2<T>{ 
	public T value;

	// 这便是泛型类中的泛型方法
	public void TestFun<K>(K k){ 
		Console.WriteLine(k);	
	}

	// 这里这个不叫泛型方法，因为T是类申明时候指定的，这里不能动态变化。
	public void TestFun(T t){ 
	
	}
}
```

## 泛型的作用
1.不同类型对象的相同逻辑处理可以选择泛型
2.使用泛型可以一定程度避免装箱拆箱

eg:优化ArrayList
```C#
class ArrayList<T>
{
    private T[] array;
    //当前数组中元素的数量
    private int count;
    //初始容量`InitialCapacity`
    private const int InitialCapacity = 4;

    public ArrayList()
    {
        array = new T[InitialCapacity];
        count = 0;
    }

    public void Add(T value)
    {
        if (count == array.Length)
        {
            // 数组已满，扩容
            T[] newArray = new T[array.Length * 2];
            Array.Copy(array, newArray, array.Length);
            array = newArray;
        }
        array[count] = value;
        count++;
    }
    public void Remove(T value)
	{
		//找到要移除元素的索引
	    int index = Array.IndexOf(array, value);
	    if (index >= 0)
	    {
		    //索引之后的元素向前移动一位
		    for (int i = index; i < count - 1; i++)
	        {
	            array[i] = array[i + 1];
	        }
	        count--;
	        //将原数组末尾的元素设置为默认值（`default(T)`）
			//确保被移除元素所占用的内存位置不会持有无效或过时的数据。
	        array[count] = default(T);
	    }
	}
}
```

## 练习
按照要求判断类型并返回类型名称和占用字节数
```C#
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine(GetTypeInfo<int>());
        Console.WriteLine(GetTypeInfo<char>());
        Console.WriteLine(GetTypeInfo<float>());
        Console.WriteLine(GetTypeInfo<string>());
        Console.WriteLine(GetTypeInfo<decimal>());
    }

    static string GetTypeInfo<T>()
    {
        Type type = typeof(T);
        if (type == typeof(int))
        {
            return "整型，4字节";
        }
        else if (type == typeof(char))
        {
            return "字符，2字节";
        }
        else if (type == typeof(float))
        {
            return "单精度浮点数，4字节";
        }
        else if (type == typeof(string))
        {
            return "字符串";
        }
        else
        {
            return "其它类型";
        }
    }
}
```
