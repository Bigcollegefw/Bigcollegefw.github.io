**1.C#中如何让自定义容器类能够使用for循环遍历？（通过 类对象[索引] 的形式遍历）**
实现索引器。
```C#
class MyList
{
    private int[] _data;
    public MyList()
    {
        _data = new int[10];
        for (int i = 0; i < 10; i++)
        {
            _data[i] = i;
        }
    }
    // 实现索引器
    public int this[int index]
    {
        get
        {
            return _data[index];
        }
    }
}
```
